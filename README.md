# File Integrity Checker (Java)
A lightweight Java utility for calculating SHA-256 file hashes.

## Compile

```bash
javac src/Main.java
```

## Run

```bash
java -cp src Main example.txt
```

## Example

```
File: example.txt
SHA-256:
7f83b1657ff1fc53b92dc18148a1d65dfa13514a...
```

## Features

- SHA-256 hashing
- Buffered file reading
- Works with files of any size
- No external dependencies
